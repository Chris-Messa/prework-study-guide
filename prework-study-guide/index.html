Prework Study Guide

HTML

  • HTML is structured using elements, constructed using tags such as the h1 tag
  • The head element contains webpage information including metadata
    this is especially important when it comes to SEO (Search engine optimization)
  • The body element represents the content that is visible to the user

CSS

  • CSS stands for cascading style sheets
  • CSS is used to style content on the web
  • CSS has many ways of selecting content on a webpage utilizing HTML elements

Git

  • Git is modern standard of code version control
  • Git allows us to create branches of our code that can later be merged to the main codebase

Javascript

  • Javascript is the programming language of the web
  • Javascript is what enables users to interact with content on the web
